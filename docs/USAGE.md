# Activity Dashboard - 使い方

## プラグインの起動

1. Obsidianを再起動するか、設定 > コミュニティプラグイン > Activity View を無効化・有効化
2. 以下のいずれかの方法でダッシュボードを開く：
   - **リボンアイコン**: 左サイドバーの棒グラフアイコンをクリック
   - **コマンドパレット**: `Ctrl/Cmd+P` → "Open Activity Dashboard"を実行

## 機能

### ヒートマップ (GitHub草風)
- 過去1ヶ月の活動をカレンダー形式で表示
- 色の濃さで活動量（入力+削除の合計）を表現
- セルにカーソルを合わせると日付と文字数を表示

### 折れ線グラフ
- 入力文字数（緑）と削除文字数（赤）の日別推移
- 2つのラインで比較可能

### 棒グラフ
- 日別の操作合計（入力+削除）を表示
- 活動量の多い日を視覚的に確認

## グラフの切り替え

ダッシュボード上部のタブボタンをクリックして切り替え：
- **ヒートマップ** 
- **折れ線グラフ**
- **棒グラフ**

## データについて

- data.jsonに保存された過去30日分のデータを自動集計
- ファイルを開く/切り替えるたびにセッションデータが記録される
- データがない場合は「データがありません」と表示

## トラブルシューティング

### ダッシュボードが表示されない
- プラグインが有効化されているか確認
- Obsidianを再起動

### グラフが空白
- data.jsonにデータがあるか確認
- 過去30日以内のデータがあるか確認
- コンソール (Ctrl+Shift+I) でエラーを確認

### Chart.jsエラー
- プラグインを再読み込み
- main.jsが正しくビルドされているか確認（209KB程度）
